## Philosophy Profile (data-only)

This folder contains a standalone question bank for **“Test Your Philosophy Profile”**.

- **No UI / logic changes** were made; this is data you can wire in later.
- **One question per weekly theme (Weeks 1–10)**, aligned with `WEEKLY_THEMES` in `app.js`.

### Files

- `questions.en.js`: English questions with A/B options and a clear value-clash axis.
